<h1>🏦 Projeto Banco Simples em Java</h1>

<p>
Este projeto consiste em um <strong>sistema bancário simples desenvolvido em Java</strong>, com foco nos conceitos iniciais de 
<strong>Programação Orientada a Objetos (POO)</strong>. Diferente da versão mais avançada (PooBank), este projeto foi criado como base de aprendizado, 
trabalhando operações essenciais de forma mais direta e didática.
</p>

<h2>🎯 Objetivo do Projeto</h2>
<p>
O objetivo principal foi entender na prática como estruturar um sistema utilizando classes, métodos e interação básica com o usuário.
</p>

<ul>
  <li>Introdução à lógica de programação em Java</li>
  <li>Manipulação de dados simples</li>
  <li>Criação e uso de classes</li>
  <li>Simulação de operações bancárias</li>
</ul>

<h2>⚙️ Funcionalidades</h2>
<ul>
  <li>Criação de conta bancária</li>
  <li>Depósito de valores</li>
  <li>Saque com validação de saldo</li>
  <li>Consulta de saldo</li>
</ul>

<h2>🏗️ Estrutura do Projeto</h2>
<ul>
  <li><strong>Conta:</strong> Classe principal contendo saldo e operações básicas</li>
  <li><strong>Main:</strong> Classe responsável pela execução do programa</li>
</ul>

<h2>🧠 Conceitos Aplicados</h2>
<ul>
  <li>Criação de classes e objetos</li>
  <li>Métodos para operações (depositar, sacar, consultar)</li>
  <li>Uso de variáveis e controle de fluxo (if/else)</li>
  <li>Validação de regras simples (ex: saldo insuficiente)</li>
</ul>

<h2>💻 Tecnologias Utilizadas</h2>
<ul>
  <li>Java (JDK 8+)</li>
</ul>

<h2>🚀 Como Executar</h2>
<pre>
1. Clone o repositório
2. Abra em uma IDE (IntelliJ, Eclipse ou VS Code)
3. Execute a classe Main.java
</pre>

<h2>📌 Evolução</h2>
<p>
Este projeto serviu como base para o desenvolvimento de uma versão mais completa do sistema bancário (PooBank), 
onde foram aplicados conceitos mais avançados como herança, polimorfismo e organização em camadas.
</p>
